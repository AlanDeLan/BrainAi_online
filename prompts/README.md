# Prompts Directory

Ця папка містить окремі файли з промптами для архетипів.

## Структура:

- `base_*.txt` - базові промпти архетипів
- `additional_*.txt` - додаткові кроки/інструкції
- Можна організувати по підпапках для різних архетипів

## Використання:

В `archetypes.yaml` можна вказати:
- `prompt_file` - шлях до файлу з основним промптом
- `additional_prompts` - список додаткових промптів (файли або текст)
- `prompt` - прямий текст (як зараз)

Все працює разом - можна комбінувати файли та текст.
