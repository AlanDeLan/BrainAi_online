# Зайві файли в проекті

## Файли, які можна видалити:

### 1. **backend/routes.py** ❌
- **Статус**: Порожній файл
- **Використання**: Не використовується ніде в проекті
- **Рекомендація**: Видалити або видалити папку `backend/` якщо вона порожня

### 2. **com.user.localbrain.plist** ❌
- **Статус**: macOS LaunchAgent конфігурація
- **Використання**: Використовується тільки на macOS для автозапуску
- **Рекомендація**: Видалити, якщо не використовуєте macOS, або перемістити в `.gitignore`

### 3. **start_local_brain.sh** ⚠️
- **Статус**: Bash скрипт для Linux/Mac
- **Використання**: Альтернатива `run_app.py` для Unix-систем
- **Рекомендація**: Залишити, якщо плануєте використовувати на Linux/Mac, або видалити якщо тільки Windows

### 4. **build/** ❌
- **Статус**: Артефакти збірки PyInstaller
- **Використання**: Створюються автоматично при збірці
- **Рекомендація**: Видалити та додати в `.gitignore`

### 5. **__pycache__/** ❌
- **Статус**: Кеш Python байт-коду
- **Використання**: Створюється автоматично
- **Рекомендація**: Видалити та додати в `.gitignore`

## Файли з невикористаними імпортами:

### 6. **main.py** - `import re` ⚠️
- **Проблема**: Імпорт `re` (regular expressions) не використовується
- **Рекомендація**: Видалити рядок `import re`

## Файли, які можна залишити (корисні утиліти):

### 7. **view_db.py** ✅
- **Статус**: Утиліта для перегляду бази даних
- **Використання**: Не використовується в основному коді, але корисна для діагностики
- **Рекомендація**: Залишити як утиліту

### 8. **run_app.spec** ✅
- **Статус**: Конфігурація PyInstaller
- **Використання**: Використовується для збірки exe файлу
- **Рекомендація**: Залишити, якщо плануєте збирати exe

## Рекомендації:

1. **Обов'язково видалити**:
   - `backend/routes.py` (або всю папку `backend/` якщо порожня)
   - `build/` (та додати в `.gitignore`)
   - `__pycache__/` (та додати в `.gitignore`)

2. **Видалити залежно від платформи**:
   - `com.user.localbrain.plist` (якщо не macOS)
   - `start_local_brain.sh` (якщо тільки Windows)

3. **Очистити код**:
   - Видалити `import re` з `main.py`

4. **Додати в `.gitignore`**:
   ```
   __pycache__/
   build/
   *.pyc
   *.pyo
   *.pyd
   .Python
   *.so
   *.egg
   *.egg-info/
   dist/
   .eggs/
   ```
